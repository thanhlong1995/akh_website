# In Hóa Đơn Nhiệt (Xprinter) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Thêm tính năng in hóa đơn thanh toán lên Xprinter (58mm/80mm) sau khi tạo đơn và từ bảng đơn gần đây, dùng `window.print()` + CSS `@media print`.

**Architecture:** Backend thêm `GET /orders/{id}` trả về chi tiết đơn kèm tên sản phẩm. Frontend thay thế popup print hiện tại bằng `#receipt` div ẩn + `@media print` CSS. Sau khi tạo đơn thành công, hiện màn success với nút "In hóa đơn"; dashboard table thêm cột in lại.

**Tech Stack:** Python FastAPI, psycopg2, HTML/CSS/Vanilla JS — không thêm dependency mới.

## Global Constraints

- UI hoàn toàn tiếng Việt
- Định dạng tiền: `320.000đ` (dấu chấm, hàm `formatCurrency()` đã có)
- Màu chủ đạo: `#185FA5`, border-radius 8px elements / 12px cards
- Flat design, không shadow, không gradient
- Không dùng JS framework
- Paper width lưu localStorage key `receipt_width`, giá trị `"58mm"` hoặc `"80mm"`, default `"80mm"`

```python
@router.get("/{order_id}", response_model=OrderDetail)
def get_order(order_id: int):
    with get_conn() as conn:
        with conn.cursor() as cur:
            cur.execute("""
                SELECT
                    o.id, o.total_amount, o.status, o.created_at,
                    c.name AS customer_name,
                    p.name AS product_name,
                    oi.quantity, oi.unit_price,
                    (oi.quantity * oi.unit_price) AS subtotal
                FROM orders o
                LEFT JOIN customers c ON o.customer_id = c.id
                JOIN order_items oi ON oi.order_id = o.id
                JOIN products p ON p.id = oi.product_id
                WHERE o.id = %s
                ORDER BY oi.id
            """, (order_id,))
            rows = cur.fetchall()
            if not rows:
                raise HTTPException(404, "Không tìm thấy đơn hàng")
            first = rows[0]
            return OrderDetail(
                id=first["id"],
                customer_name=first["customer_name"],
                total_amount=float(first["total_amount"]),
                status=first["status"],
                created_at=first["created_at"],
                items=[
                    OrderItemDetail(
                        product_name=row["product_name"],
                        quantity=float(row["quantity"]),
                        unit_price=float(row["unit_price"]),
                        subtotal=float(row["subtotal"]),
                    )
                    for row in rows
                ]
            )
```

Tìm và xóa toàn bộ hàm `printOrder()` cũ (từ `// ── Print order` line 458 đến hết dấu `}` đóng của hàm ở line 565).

```javascript
// ── In hóa đơn ─────────────────────────────────────────────────────────────

async function printOrder(orderId) {
  try {
    const order = await api(`/orders/${orderId}`);

    // Populate header
    const dt = new Date(order.created_at);
    document.getElementById('receipt-datetime').textContent =
      dt.toLocaleString('vi-VN', { day: '2-digit', month: '2-digit', year: 'numeric',
                                    hour: '2-digit', minute: '2-digit' });
    document.getElementById('receipt-order-id').textContent = `Hóa đơn #${order.id}`;
    document.getElementById('receipt-customer').textContent =
      order.customer_name ? `Khách: ${order.customer_name}` : 'Khách vãng lai';

    // Populate items
    document.getElementById('receipt-items-body').innerHTML =
      order.items.map(i => `
        <tr>
          <td>${escapeHtml(i.product_name)}</td>
          <td class="receipt-r">${i.quantity}</td>
          <td class="receipt-r">${formatCurrency(i.unit_price)}</td>
          <td class="receipt-r">${formatCurrency(i.subtotal)}</td>
        </tr>`).join('');

    // Populate total
    document.getElementById('receipt-total').textContent =
      `TỔNG: ${formatCurrency(order.total_amount)}`;

    // Inject paper width
    const width = localStorage.getItem('receipt_width') || '80mm';
    let styleEl = document.getElementById('receipt-page-style');
    if (!styleEl) {
      styleEl = document.createElement('style');
      styleEl.id = 'receipt-page-style';
      document.head.appendChild(styleEl);
    }
    styleEl.textContent = `@media print { @page { size: ${width} auto; } }`;

    window.print();
  } catch (err) {
    showToast(`Không thể in: ${err.message}`, true);
  }
}
```
